# Hotel Management & Analytics — Frontend

React 18, Vite 5, Tailwind CSS, React Router.

## Chạy dev

1. Cài dependency:
   ```bash
   npm install
   ```
2. Khởi động dev server:
   ```bash
   npm run dev
   ```
3. Backend FastAPI mặc định tại **http://127.0.0.1:8000**. Có thể đổi API base bằng biến môi trường:
   ```bash
   # .env.local
   VITE_API_BASE_URL=http://127.0.0.1:8000/api/v1
   ```

## Build & preview

```bash
npm run build
npm run preview
```
